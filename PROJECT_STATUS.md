## 🚀 启动说明

### 快速启动（推荐）
使用项目根目录的启动脚本：
```bash
start.bat
```

### 手动启动
1. **启动后端**
   ```bash
   cd backend
   mvn spring-boot:run
   ```

2. **启动前端**
   ```bash
   cd frontend
   npm run dev
   ```

### 访问地址
- 前端应用: http://localhost:5173
- 后端API: http://localhost:8080/api
